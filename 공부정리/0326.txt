# Java 프로그래밍 기초
 - 자료형 : 데이터의 종류, 메모리 크기, 데이터의 표현 범위
   기본 자료형 - primity type
   - 정수형 : byte - char - short - int - long
   - 실수형 : float - double
     - 논리형 : boolean
  참조 자료형 - reference type

 - 연산자
   대입연산자, 복합 대입 연산자, 관계 연산자, 논리 연산자, 증감 연산자, 조건 연산자(삼항)
 
 - 형변환
   자동 형변환(묵시적), 강제 형변환(명시적)

 - scanner
