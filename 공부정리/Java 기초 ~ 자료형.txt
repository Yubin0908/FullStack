# Java 프로그래밍 기초
 - 자료형 : 데이터의 종류, 메모리 크기, 데이터의 표현 범위
   기본 자료형 - primity type
    - 정수형 : byte - char(문자) - short - int(기본) - long
    - 실수형 : float - double(기본)
    - 논리형 : boolean
   참조 자료형 - reference type

 - literal(리터럴)
  - 그 자체의 값을 의미.
  - 이름 없음.
  - 접미사 : Long(L), Float(F)
  - 정수형 리터럴 : int
  - 실수형 리터럴 : double
  - 논리형 리터럴 : boolean
  - 문자형 리터럴 : int 
  - 문자열 리터럴 : String

- 상수 (constant) - 변하지 않는 값
